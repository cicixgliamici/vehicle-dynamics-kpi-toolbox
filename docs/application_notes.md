# Application notes

This MVP is designed to support an application for MATLAB-based vehicle dynamics data processing roles.

## What it demonstrates

- MATLAB scripting
- Modular project organization
- Signal preprocessing
- KPI extraction
- Validation-oriented thinking
- Automated reporting
- Basic vehicle dynamics vocabulary

## How to describe it in a CV

> Developed a MATLAB vehicle dynamics data-processing toolbox for synthetic test-track and simulation data, including signal preprocessing, configurable KPI extraction, maneuver detection, validation tests, and automated plotting/reporting for steering, handling, yaw-response and ride-comfort analysis.

## Future v1/v2 improvements

- Add a linear bicycle model.
- Add `ode45` integration.
- Add parameter sweeps over vehicle parameters.
- Add noisy sensor-data generation.
- Add simulation-vs-measurement KPI validation.
